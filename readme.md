# EnergyGreen - Landing Page Sustentável

## Sobre o Projeto
O **EnergyGreen** é uma landing page corporativa focada em fontes de energia renováveis, sustentabilidade e soluções ESG. O projeto apresenta tecnologias voltadas para um mundo melhor, destacando energias limpas como a eólica, solar e hidrelétrica.

Este layout foi estruturado como aplicação prática de Front-end para o curso de Análise e Desenvolvimento de Sistemas (ADS) da FIAP, focando na construção de interfaces limpas e alinhadas às demandas do mercado corporativo atual.

##  Funcionalidades
*   **Hero Banner:** Imagem de destaque imersiva com tipografia de alto contraste para prender a atenção do usuário logo no primeiro contato.
*   **Grid de Soluções:** Apresentação organizada em cartões (Cards) exibindo as diferentes vertentes de energia limpa através de CSS Grid/Flexbox.
*   **Identidade Visual:** Paleta de cores baseada em tons de verde escuro e imagens sóbrias para transmitir seriedade, credibilidade e foco ambiental.

##  Tecnologias Utilizadas
*   **HTML5:** Estruturação semântica e organização de blocos de conteúdo.
*   **CSS3:** Estilização em cascata, alinhamento avançado e responsividade visual.
